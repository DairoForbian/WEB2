JSON-LD

JSON-LD (JSON for Linking Data) — формат для встраивания структурированных данных в HTML-страницы, рекомендованный Google. Разметка помещается в тег <script type="application/ld+json"> и не смешивается с контентом.

Преимущества

  • Не требует изменения HTML-структуры.
  • Легко добавляется и редактируется.
  • Поддерживает все типы Schema.org.
  • Можно автоматически генерировать из CMS.

Пример разметки для статьи:

{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "...",
  "author": { "@type": "Person", "name": "..." },
  "datePublished": "...",
  "image": "..."
}

Проверять разметку можно в Google Rich Results Test.
