О проекте

Учебный проект в рамках курса «Технологии разработки интернет-приложений». Цель — закрепить навыки создания семантической, адаптивной и доступной вёрстки с использованием современного CSS‑фреймворка и веб‑компонентов.

Тематика

Практическое руководство по поисковой оптимизации (SEO): техническое SEO, семантический HTML, структурированные данные Schema.org и формат JSON-LD.

Использованные технологии

  • HTML5, CSS3
  • CSS‑фреймворк: MVP.css
  • JavaScript (ES6+) – веб‑компоненты
  • Git и GitHub Pages

Контент страниц основан на открытых источниках и официальной документации соответствующих технологий.
